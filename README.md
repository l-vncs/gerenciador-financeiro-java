# Gerenciador Financeiro em Java

##  Descrição
Sistema de controle financeiro via terminal, permitindo cadastro de receitas e despesas, cálculo de saldo e persistência de dados.

## Funcionalidades
- Cadastro de receitas
- Cadastro de despesas
- Cálculo automático de saldo
- Listagem completa
- Salvamento em arquivo

##  Tecnologias
- Java
- Programação Orientada a Objetos
- Manipulação de Arquivos

## Aprendizados
- Herança e polimorfismo
- Organização em camadas (Model / Service)
- Persistência de dados
